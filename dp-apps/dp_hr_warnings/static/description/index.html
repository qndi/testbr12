Dieses Modul ist eine Erweiterung für die dp Time Report App

Das dp Time Report Modul integriert 5 verschiedene Odoo Apps in eine einzige App. Dabei wurde ein besonderes Augenmerk darauf gelegt, dass die Odoo Standards so weit wie möglich beibehalten werden. Dies erhöht die Benutzerfreundlichkeit und bietet durch Erweiterungen noch zusätzliche Benefits.

Warnungen

Diese Erweiterung unterstützt bei der Einhaltung arbeitsrechtlicher Vorgaben. Es können Warnungen im Bezug auf Wochenendarbeit, arbeiten über ein Maximum an Stunden, arbeiten ohne Pause uvm. definiert werden. Sollte ein Mitarbeiter solch eine Buchung durchführen, erscheinen im Report Warnungen.
